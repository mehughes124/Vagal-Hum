# Vagal-Hum
Vagal Hum is a simple tone generator for humming practice.  Tap the orb. A clean sine wave plays. Hum along, matching the pitch. Feel for the buzz in your chest and throat — that's vagus nerve stimulation via the laryngeal and pharyngeal branches that innervate those structures.  A breath guide paces you: 4 seconds inhale, 8 seconds hum-out.
